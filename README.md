# KNN Classifier

This repository contains a K-Nearest Neighbors (KNN) classification workflow built with Python and scikit-learn.

## Contents
- Data loading and exploration
- Feature scaling using `StandardScaler`
- Train-test split
- KNN model training and prediction
- Model evaluation using confusion matrix and classification report
- K-value selection using error rate analysis

## Tools and Libraries
- Python
- pandas
- NumPy
- matplotlib
- seaborn
- scikit-learn

## Goal
The goal of this project is to build and evaluate a KNN classification model and identify a suitable K value for better performance.
